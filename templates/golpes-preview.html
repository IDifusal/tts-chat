¿Cómo agregar skins / modelos propios?

Lo que usa skinview3d

skinview3d acepta PNG 64×64 en formato estándar de Minecraft (textura con las secciones de cabeza, cuerpo, brazos y piernas desplegadas). No soporta modelos 3D custom (OBJ, GLTF, etc.).

Para cambiar la skin de este demo

  1. Coloca tu PNG de skin (64×64) en static/pelea/skins/minombre.png
  2. Abre este HTML y edita la línea del JS:
    var SKIN_URL = base + '/static/pelea/skins/chupetin.png';
    → cambia chupetin por el nombre de tu archivo.
  3. Para P1 y P2 con skins distintas, define SKIN_P1 y SKIN_P2 y pasa cada uno al viewer correspondiente.

Para modelos 3D completamente custom (no Minecraft)

Habría que reemplazar skinview3d por Three.js puro, cargar un archivo .glb/.gltf con GLTFLoader y animar con AnimationMixer. La lógica de pelea (golpe, esquivar, resultado) permanecería igual; solo cambia la parte de renderizado.

Dónde obtener skins 64×64

  • minecraftskins.com → descargar PNG (no la preview)
  • namemc.com → buscar jugador → Download Skin
  • minotar.net/skin/USERNAME → skin directo por nick
  • skindex.com → miles de skins descargables
